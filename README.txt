Workflow

1. Use solveRoutes.py to solve all your routes in your stops. 
2. Merge the routes with joinRoutes.py 
3. Join the newly merged routes from joinRoutes.py to geographical zones. 
4. Display choropleth map with desired field attributes. 

Requires: 
1. Working knowledge of ArcGIS Network Analystr 10.1+.
2. Pre-processed stops that already lie within transportation networks.
3. Pre-processed network dataset with desired parameters and restrictions.
